# הלכות קידוש החודש — אפליקציית לימוד אינטראקטיבית

## סקירת הפרויקט

אפליקציית לימוד אינטראקטיבית להלכות קידוש החודש של הרמב"ם (משנה תורה), עם ביאור שטיינזלץ, תרגום אנגלי (Touger), והמחשות ויזואליות.

### מצב נוכחי: MVP עובד בקובץ HTML יחיד (576KB)

---

## ארכיטקטורה

### קובץ אחד: `kiddush_hachodesh_unified.html`
- HTML + CSS + JS הכל inline
- נתוני 235 הלכות (JSON) מוטמעים בתוך `<script>`
- Three.js נטען מ-CDN
- אין צורך בשרת — עובד כקובץ סטטי

### מבנה הנתונים (`const DATA`)
```js
[
  {
    ch: 1,  // מספר פרק
    halachot: [
      {
        n: 1,           // מספר הלכה
        he: "...",       // טקסט רמב"ם עברית (תורת אמת / ספריא)
        en: "...",       // תרגום אנגלי (Touger / Moznaim)
        st: ["...", ...],// ביאור שטיינזלץ (מערך פסקאות)
        viz: "moon3d"    // סוג המחשה (אופציונלי, רק ל-22 הלכות)
      }
    ]
  }
]
```

### כותרות (`const CH_TITLES`, `const HAL_TITLES`)
```js
CH_TITLES = { 1: "חודשי לבנה ושנות חמה", ... }  // 19 פרקים
HAL_TITLES = { "1:1": "חודשי השנה — חודשי הלבנה", ... }  // 235 הלכות
```

## 3 מצבי תצוגה

### 🔭 סקירה (Overview)
- 4 חלקים (layers) מתקפלים
- כל חלק עם ניווט שלבים (step dots)
- ~24 שלבי לימוד עם הסברים + המחשות
- כולל מודל Three.js של ירח, מחשבונים, טבלאות

### 📖 לימוד (Study)
- ניווט לפי פרק (סרגל אופקי נגלל)
- הלכות באקורדיון — לחיצה פותחת
- כל הלכה: טקסט רמב"ם → שטיינזלץ → המחשה (אם יש)

### 📜 רציף (Continuous)
- כל 235 הלכות בגלילה אחת
- חלוקה לפרקים עם כותרות
- מצביע פרק צף (floating progress)
- אפס לחיצות — פשוט גוללים
